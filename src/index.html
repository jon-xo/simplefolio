About me

I live in Southern West Virginia, bordering the New River Gorge National Park. After growing up in the heart of Philadelphia, seven years ago, I decided to make a change. I now live on a working family farm. Since moving, I’ve had the opportunity to build a house, practice animal husbandry and operate heavy equipment.

Over the last ten years of my career, I have worked for Apple. I started as an Apple Store Genius in Philadelphia and later moved departments and joined the AppleCare Enterprise team as a remote senior advisor. Throughout my experience, I was able to work as a technician, server administrator, subject-matter expert, and technical writer. In 2020, I decided to make a career shift into software development by attending Generation WV’s NewForce program. During this time, I rediscovered the joy of learning new technologies and applying my technical knowledge to solve everyday problems.

I love hiking, cycling, motorcycling, and running after with my kids. I have a deep love of vintage video games and am teaching myself how to fix and restore arcade cabinets.

Projects

bifröst - Community Comic Book Manager

Bifröst is a single-page comic book management application that combines new comic book discovery with an integrated social, user-sharing platform. Inspired by the community-based reading application Goodreads, bifröst allows users to curate a personal reading or pull-list of comics and share these selections in a continuous public timeline.

Bifröst uses Google Firebase for user authentication and is built with a React frontend. The backend is constructed using an ASP.NET Core API and Microsoft SQL Server database.

casCade - Classic Arcade App

CasCade is a classic arcade single-page application. This application provides an intuitive front-end layout powered by Archive.org’s Internet Arcade collection. I made this application to provide an easy-to-use, inviting design that allows users to access and play their arcade favorites.

CasCade was built using React, RESTful APIs via json-server, notistack notification library, and styled with Material-UI.

dot-back

Dot-back is a shell script that copies a user's dot-files to a local, version-controlled directory. This project ensures a known working copy of a dot file matched from a preconfigured list can be kept safe.

Once executed, dot files present in a User's home directory are copied to the directory corresponding to the current Operating System. Additionally, a cron job is created to schedule the script in the future.
